# 0x0B. C - Preprocessor

## Description
This directory contains programs and header files that demonstrate
the use of the C preprocessor, macros, and structures.

## Files

### 0-object_like_macro.h
A header file that defines a macro named SIZE as an abbreviation
for the token 1024.

## Compilation
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-main.c -o a

## Usage
./a
