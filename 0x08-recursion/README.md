# 0x08. C - Recursion

This project covers recursion in C: writing functions that call themselves
to solve a problem, using a base case to stop and a recursive case to
keep progressing toward it.

## Files

- `0-puts_recursion.c` - prints a string, followed by a new line, using recursion
- `1-print_rev_recursion.c` - prints a string in reverse, using recursion
